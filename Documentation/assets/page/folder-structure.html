      Folder Structure

      Insights into Application Directory

      Upon completion of the installation, your main folder will host two directories and two files.

      • assets
      • main
      • .htaccess
      • index.php
      • In the assets folder, we store various assets such as CSS, JS, Images. If you wish to modify or update anything related to CSS or JS, kindly refer to this directory.
      • In the main folder, the essence of Laravel is preserved, maintaining a comprehensive MVC structure throughout the entirety of this project.
