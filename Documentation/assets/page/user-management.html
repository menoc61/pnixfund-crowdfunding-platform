      User Management

      Central Hub for User Administration

      In our Admin Panel's User Management section, experience a dynamic suite of tools that puts you in command of user administration. From a detailed user list to individual account details, and from managing balances to overseeing transaction counts, our platform offers unparalleled control.

      • User List :
        Access a detailed list of all registered users, providing a comprehensive overview of your user base.
      • Searching Users :
        Admins can search for users by their unique usernames or emails. Admins also have the flexibility to conduct searches within a specified range of joining dates. This powerful feature allows for the identification of users who registered during a particular timeframe.
      • User Details :
        Dive into individual user profiles, exploring their transaction history, preferences, and account information.
      • Balance Overview :
        Monitor user balances at a glance, ensuring and understanding of their financial status.
      • Transaction Counts :
        Track exchange and withdrawal counts, gaining valuable insights into user activity.
      • Login as User :
        Seamlessly log in as a user for efficient troubleshooting or support, enhancing your user assistance capabilities.
      • Balance Updates :
        Effortlessly adjust user balances as needed, ensuring accurate financial records.
      • Ban User :
        Maintain control over user access by implementing user bans when necessary, preserving the integrity of your platform.
      • User Details Update :
        Edit and update user details as needed, ensuring accurate and up-to-date information.
      • Email Verification :
        Manage email verification status, enhancing security measures for user accounts.
      • Mobile Verification :
        Monitor mobile verification status, adding an extra layer of authentication to user profiles.
      • Two-Factor Authenticator :
        Enable or update two-factor authentication settings, bolstering the security of user accounts.

      Navigate through these editable features with ease, customizing user details and account settings to meet the specific needs of your crowdfunding platform. Explore the flexibility of User Management and take control of your platform's user experience.
