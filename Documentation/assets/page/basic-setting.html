      Basic Setting

      Administrative Site Control Hub

      The Basic Setting Management System within our Admin Panel serves as a pivotal tool for administrators, allowing them to fine-tune and customize essential parameters that significantly impact the overall user experience. This feature-rich system comprises three key segments, each playing a crucial role in shaping the platform's identity and functionality.

      Site Preferences:
      • Site Name :
        Administrators can set and modify the site name, establishing a distinct and recognizable identity for the crowdfunding platform.
      • Platform Base Currency :
        Define the base currency for the entire platform, influencing currency conversions and transactions.
      • Time Region :
        Set the application's time zone, ensuring accurate and synchronized timing across the entire system.
      • Site Color :
        Choose and customize primary and secondary colors that will harmonize the platform's visual identity, providing a cohesive and appealing user interface.

      System Preferences:
      • User Signup :
        Control user registration by enabling or disabling the signup feature based on current requirements.
      • Enforce Strong Password :
        Enhance security by enforcing users to make strong password, promoting user account protection.
      • Accept Policy :
        Set the toggle for policy acceptance during user registration, ensuring compliance with terms and conditions.
      • KYC Check :
        Implement or disable KYC checks as needed, aligning with regulatory requirements.
      • Email Confirmation :
        Choose whether users are required to confirm their email addresses, ensuring valid and verified user accounts.
      • Email Alert :
        Enable or disable email alerts for various system notifications, providing flexibility in user communication.
      • Mobile Confirmation :
        Choose whether users need to confirm their mobile numbers, enhancing account security.
      • SMS Alert :
        Control the activation of SMS alerts for users, facilitating efficient communication.
      • Enforce SSL :
        Enhance platform security by enforcing SSL connections for encrypted data transmission.
      • Language Activation :
        Enable or disable language, allowing for a tailored multilingual user experience.

      Logo & Favicon Preferences:
      • Light and Dark Logo :
        Customize the platform's appearance with distinct light and dark versions of the site logo.
      • Favicon Icon :
        Set the favicon icon, contributing to brand recognition and user experience.

      This comprehensive Basic Setting Management System provides administrators with the tools needed to shape the platform's identity, functionality, and user interaction. From visual aesthetics to security measures and language preferences, this system empowers administrators to create a tailored and impactful crowdfunding platform.
